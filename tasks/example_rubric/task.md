# Example task — landing-page rewrite (rubric-scored)

**Goal:** Given a dry, technical sentence from an engineering-spec doc, rewrite it
as a single sentence that would feel at home on a modern SaaS product landing
page — friendly, benefit-forward, plain English, no jargon, no marketing fluff,
still factually faithful to the original.

Output only the rewritten sentence. No preamble, no quotes, no Markdown.
